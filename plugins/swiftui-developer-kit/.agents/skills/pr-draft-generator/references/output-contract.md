# Required Quality Fields

For each factual finding, include severity, evidence, user impact, concrete fix or next action, verification, confidence, and missing evidence. For PR planning without a defect, state scope, risk or status, evidence, next action, verification, confidence, and missing evidence. State safety, privacy, approval, and not-inspected boundaries where relevant.

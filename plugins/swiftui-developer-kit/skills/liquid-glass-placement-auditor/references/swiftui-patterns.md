# SwiftUI Patterns To Inspect

Search patterns:

```text
NavigationStack
NavigationSplitView
TabView
.toolbar
.sheet
.popover
.inspector
safeAreaInset
.background
.ultraThinMaterial
.thinMaterial
.regularMaterial
.opacity
.accessibilityLabel
```

Review custom controls:

- Sidebars
- Bottom bars
- Floating action buttons
- Canvas tool palettes
- Apple Pencil controls
- Search bars
- PDF viewers
- Flashcard views
- Settings forms
- Dense note or planner lists

Look for custom blur, hardcoded opacity, and hardcoded colors that may conflict with system materials.

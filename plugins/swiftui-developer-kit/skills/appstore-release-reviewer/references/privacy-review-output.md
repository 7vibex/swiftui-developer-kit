# Privacy Review Output

For every data category, report:

- Data type and source.
- Whether it is collected, stored locally, exported by the user, or uploaded.
- Purpose and retention.
- Linkage to identity and tracking behavior.
- Redaction or minimization.
- `PrivacyInfo.xcprivacy` declaration and App Store Connect answer.
- Evidence inspected and evidence missing.

Treat diagnostics, crash reports, analytics, support attachments, AI prompts, handwritten content, and document names as separate flows. A local schema does not prove remote behavior.

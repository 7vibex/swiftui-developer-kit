Codex SwiftUI Developer Kit

A focused skill pack for auditing, debugging, modernizing, testing, screenshot-reviewing, and releasing SwiftUI Apple apps.

audit
Review architecture, persistence, UI, accessibility, and release risks.
canvas-audit
Inspect PencilKit, zoom/pan, gestures, layers, persistence, and canvas performance.
fix-build
Diagnose Xcode, scheme, package, simulator, and signing failures.
review-screenshots
Capture only with approval, then inspect layout and visual hierarchy.
modernize-ui
Review Apple UI fit, SwiftUI structure, and where Liquid Glass belongs.
detect-risks
Run deterministic SwiftUI source checks before deeper review.
prepare-release
Check TestFlight, App Store, privacy, metadata, screenshots, and signing.

Try The Local CLI

scripts/swiftui-kit.sh list
scripts/swiftui-kit.sh detect --format markdown .
scripts/swiftui-kit.sh doctor

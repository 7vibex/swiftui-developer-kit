# Appshots Workflow

Appshots can capture the frontmost app window for visual review. Use them only after explicit permission.

## User Setup

The user may need to:

- Grant macOS Screen Recording permission.
- Grant Accessibility permission if the tool needs window interaction.
- Bring the correct app or Simulator window to the front.
- Close or hide sensitive windows.

## Safety Rules

- Never capture passwords, private chats, personal documents, production secrets, tokens, API keys, or customer data.
- Ask before every capture session.
- Confirm the intended window is frontmost.
- Use screenshot filenames that describe the screen and order.

## Review Use Cases

- Liquid Glass placement review.
- Layout, spacing, clipping, and hierarchy checks.
- Dynamic Type and readability comparison.
- Before/after visual verification.

# v1.0.0 - Trust Release

Status: release criteria, not a claim that v1.0.0 has shipped.

## Required Evidence

- Stable public fixture and output suite.
- Reproducible benchmark results.
- Threat model and privacy-safe diagnostics schema.
- Strict output contract, evidence standard, boundaries, and paired examples for all skills.
- Triaged remote issue tracker and versioned release notes.

## Acceptance

- A new user can install, validate, scan fixtures, and inspect skill output in under ten minutes.
- Limitations are visible before any marketing claims.
- No release claim depends on private project evidence.

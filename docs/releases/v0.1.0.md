# v0.1.0 - Initial SwiftUI Codex Skill Pack

Initial public release of Codex SwiftUI Developer Kit.

Release date: June 1, 2026.

## Highlights

- 12 Codex skills for SwiftUI, SwiftData, Xcode, accessibility, Simulator screenshots, Liquid Glass placement, release review, testing, and PR drafting.
- Router skill for broad Apple app requests.
- Consent-first screenshot workflows.
- Safe shell scripts for project detection, file collection, screenshot capture, and local installation.
- Local installer that symlinks skills into `$HOME/.agents/skills`.
- Validation script and GitHub Actions workflow for skill frontmatter, references, scripts, examples, and safety checks.
- README demo showing the prompt-to-skill-to-result workflow.
- AI Usage Policy covering reviewed AI assistance and unsafe automation boundaries.
- Public, anonymized proof points from real SwiftUI architecture and Liquid Glass placement audits.

## Known Limits

- Example reports use a fictional StudyOS-style app.
- Appshots and Computer Use availability depends on the user's Codex environment.
- The OpenAI Codex documentation links should be rechecked as public Codex docs evolve.
- Screenshot-backed confidence requires user-approved Simulator capture; code-only UI audits are marked with appropriate confidence.
